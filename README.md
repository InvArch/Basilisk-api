# HydraDX-api
## Description
An API to satisfy your most hydrated needs.

## Running

### Dev
`npm run dev`

### Test
`npm run test`

### Production
`npm run start`

### Docker
`docker build -t hydradx-api .`
`docker run -p 3000:3000 -d hydradx-api`

## API Framework
Built using Fastify, check out the [Fastify documentation](https://www.fastify.io/docs/latest/).
